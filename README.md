# Language Learning Flashcards Advanced

A flashcard web application for checking English word translations with different difficulty levels.

## Live Demo

https://alexandrabalinskaiya.github.io/language-learning-flashcards-advanced/

## Description

Language Learning Flashcards Advanced is a frontend practice project built with HTML, CSS, JavaScript and jQuery.

The application shows random English words, and the user needs to enter the correct Ukrainian translation. The user can choose a difficulty level before starting the test. The app tracks correct and incorrect answers, shows the current step, and displays the final language level after completing all questions.

## Features

* random word selection
* difficulty level selection
* flashcard interface
* translation input validation
* correct and incorrect answer counter
* step counter
* final result modal
* automatic test reset
* responsive layout

## Technologies

* HTML5
* CSS3
* JavaScript
* jQuery

## How to Use

1. Open the live demo.
2. Select a difficulty level.
3. Look at the English word on the flashcard.
4. Enter the Ukrainian translation.
5. Click the Check button.
6. Complete 10 steps.
7. The application will show your result level.
